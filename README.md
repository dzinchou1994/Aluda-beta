# AludaAI - AI рЃљрЃАрЃўрЃАрЃбрЃћрЃюрЃбрЃў рЃЦрЃљрЃарЃЌрЃБрЃџ рЃћрЃюрЃљрЃќрЃћ

AludaAI рЃљрЃарЃўрЃА Next.js 14-рЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў AI рЃљрЃАрЃўрЃАрЃбрЃћрЃюрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃюрЃбрЃћрЃњрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ Flowise-рЃЌрЃљрЃю рЃЊрЃљ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃЦрЃљрЃарЃЌрЃБрЃџ рЃћрЃюрЃљрЃќрЃћ.

## Рюе рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў

- ­Ъцќ **Flowise рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ** - рЃърЃўрЃарЃЊрЃљрЃърЃўрЃарЃў API рЃЎрЃљрЃЋрЃерЃўрЃарЃў
- ­ЪїЇ **рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃћрЃюрЃљ** - рЃЏрЃЌрЃџрЃўрЃљрЃюрЃў UI рЃЦрЃљрЃарЃЌрЃБрЃџ рЃћрЃюрЃљрЃќрЃћ
- ­Ъћљ **рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ** - Google, Apple рЃЊрЃљ Guest рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
- ­Ъњг **ChatGPT-рЃўрЃА рЃЏрЃАрЃњрЃљрЃЋрЃАрЃў рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў** - рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃў рЃЊрЃљ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў
- ­ЪЊ▒ **Responsive Design** - Tailwind CSS + shadcn/ui
- ­Ъџђ **Next.js 14 App Router** - рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў

## ­Ъџђ рЃАрЃгрЃарЃљрЃцрЃў рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљ

### 1. рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ

```bash
npm install
```

### 2. рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ `.env.local` рЃцрЃљрЃўрЃџрЃў:

```bash
cp env.example .env.local
```

рЃЊрЃљ рЃерЃћрЃљрЃЋрЃАрЃћрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў:

```env
# Flowise Integration
FLOWISE_HOST=https://flowise-eden.onrender.com
FLOWISE_CHATFLOW_ID=9a1520d5-52e3-4365-9c47-27c89f40ddeb
FLOWISE_API_KEY=your-api-key-here

# Database
DATABASE_URL="postgresql://username:password@localhost:5432/aluda_ai"

# Authentication
JWT_SECRET=your-super-secret-jwt-key-here
NEXTAUTH_SECRET=your-nextauth-secret-here
NEXTAUTH_URL=http://localhost:3000
```

### 3. рЃЉрЃљрЃќрЃўрЃА рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃљ

```bash

npx prisma db push
```

### 4. рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃЏрЃћрЃюрЃбрЃўрЃА рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ


рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ [http://localhost:3000](http://localhost:3000) рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў.

## ­ЪћД рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

### Flowise рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ

AludaAI рЃўрЃДрЃћрЃюрЃћрЃЉрЃА Flowise-рЃўрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа API-рЃА:
- рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃћрЃю `/api/chat` endpoint-рЃА
- рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃЏрЃ«рЃљрЃарЃћрЃА рЃърЃарЃЮрЃЦрЃАрЃў Flowise-рЃЌрЃљрЃю
- рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃЊрЃљ рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃўрЃарЃћрЃЉрЃљрЃЊрЃў

### рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў

| рЃфрЃЋрЃџрЃљрЃЊрЃў | рЃљрЃдрЃгрЃћрЃарЃљ | рЃЊрЃћрЃцрЃЮрЃџрЃбрЃў |
|---------|---------|----------|
| `FLOWISE_HOST` | Flowise рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА URL | - |
| `FLOWISE_CHATFLOW_ID` | Chatflow ID | - |
| `FLOWISE_API_KEY` | API рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃў (рЃЮрЃцрЃфрЃўрЃЮрЃюрЃљрЃџрЃБрЃарЃў) | - |

## ­ЪЊ▒ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ

### рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЕрЃљрЃбрЃў
1. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ `/chat` рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ
2. рЃЊрЃљрЃгрЃћрЃарЃћрЃЌ рЃерЃћрЃЎрЃўрЃЌрЃ«рЃЋрЃљ рЃЦрЃљрЃарЃЌрЃБрЃџ рЃћрЃюрЃљрЃќрЃћ
3. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ Enter рЃљрЃю Send рЃдрЃўрЃџрЃљрЃЎрЃА
4. рЃЏрЃўрЃўрЃдрЃћрЃЌ AI-рЃўрЃА рЃърЃљрЃАрЃБрЃ«рЃў

### рЃЎрЃџрЃљрЃЋрЃўрЃљрЃбрЃБрЃарЃўрЃА рЃерЃЮрЃарЃбрЃЎрЃљрЃбрЃћрЃЉрЃў
- `Enter` - рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃўрЃА рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃљ
- `Shift + Enter` - рЃљрЃ«рЃљрЃџрЃў рЃ«рЃљрЃќрЃў

## ­Ъћњ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃљ

- **Rate Limiting** - 10 рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ рЃгрЃБрЃЌрЃерЃў
- **Session Management** - рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ cookie-рЃћрЃЉрЃў
- **CORS** - рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЊрЃљрЃерЃЋрЃћрЃЉрЃБрЃџрЃў origins
- **Input Validation** - рЃДрЃЋрЃћрЃџрЃљ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ
- **Error Handling** - рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃљрЃџрЃЋрЃљ

рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃљрЃњрЃќрЃљрЃЋрЃюрЃўрЃА рЃЏрЃўрЃюрЃўрЃЏрЃљрЃџрЃБрЃа рЃбрЃћрЃџрЃћрЃЏрЃћрЃбрЃарЃўрЃљрЃА:
